# Drive Kargo

Use this skill when defining Kargo Projects/Warehouses/Stages, promoting Freight, or authenticating with the Kargo CLI.

## Quick start
1. Log in with the Kargo CLI (Drive SSO or admin credentials).
2. Create or apply Project/Warehouse/Stage resources.
3. Promote Freight through Stages in the UI.
